## 📊 1. Creating PowerPoint from PITCH-DECK-SLIDES.txt

### Option A: Manual (Best Quality)

**Software needed:** PowerPoint, Keynote, or Google Slides

**Steps:**
1. Open PowerPoint/Keynote/Google Slides
2. Create new presentation
3. Set theme:
   - Font: Playfair Display (titles), Inter (body)
   - Colors: Forest Green (#1F4E3D), Sunset Orange (#F59E0B)
4. Copy content from PITCH-DECK-SLIDES.txt slide by slide
5. Add charts/graphs where indicated
6. Insert artistrax logo on each slide (top right)
7. Export as PDF for sharing

**Time:** 2-3 hours for professional quality

## 💰 3. Using Financial Model Spreadsheet

### Import to Excel

**Steps:**
1. Open Excel
2. Click "File" → "Open"
3. Select `artistrax-financial-model.csv`
4. Excel will import automatically
5. Save as .xlsx file

**Formatting:**
1. Format currency columns (add $ sign)
2. Format percentages (add %)
3. Add colors to headers (green for revenue, red for costs)
4. Create charts from data

---

### Import to Google Sheets

**Steps:**
1. Go to sheets.google.com
2. Click "Blank" to create new sheet
3. Click "File" → "Import"
4. Click "Upload" tab
5. Drag `artistrax-financial-model.csv` file
6. Choose "Replace spreadsheet"
7. Click "Import data"
8. Done!

**Formatting:**
1. Select column B-F (numbers)
2. Click "Format" → "Number" → "Currency"
3. Select percentage rows
4. Click "Format" → "Number" → "Percent"
5. Add conditional formatting (green for profit, red for loss)

---

### Create Charts

**Recommended charts:**

**Chart 1: Revenue Growth (Line chart)**
- X-axis: Years (2026-2030)
- Y-axis: Total Revenue
- Shows hockey stick growth

**Chart 2: Profit Margin (Column chart)**
- X-axis: Years
- Y-axis: Profit Margin %
- Shows profitability

**Chart 3: User Growth (Stacked area)**
- X-axis: Years
- Y-axis: Users
- Layers: Artists, Labels, Fans

**Chart 4: Revenue Mix (Pie chart)**
- Subscription vs Platform Fees
- Shows 90% is subscription

## 🎨 4. Creating Graphics in Canva

4. Choose blank canvas (1080x1080)

### Quote Card Template

**Instructions from materials:**
1. Set background: Forest green gradient (#1F4E3D to #0F2E1D)

4. Add logo (bottom right, white)
5. Export as PNG

**Time:** 5 minutes

---

### Stat Card Template

1. Create 1080x1080 canvas
2. Draw rectangle: Left half green (#1F4E3D), right half orange (#F59E0B)
3. Add text "95%" on left (huge, white, bold)
4. Add text "to artists on every sale" on right (smaller, white)
5. Add logo bottom center
6. Export as PNG

**Time:** 5 minutes

---

### Artist Spotlight Template

1. Create 1080x1080 canvas
2. Upload artist photo
3. Crop to circle, center at top
4. Add text:
   - Artist name (bold, 36pt)
   - Genre | Location (16pt)
   - Bio (18pt, 2-3 sentences)
   - "Listen now: artistrax.com/username" (20pt, green)
5. White background with green accents
6. Export as PNG

**Time:** 10 minutes

## 🎯 Success Metrics (Track Weekly)

### Social Media:
- Followers (growth rate)
- Engagement rate (likes + comments / followers)
- Link clicks (to artistrax.com)
- Conversions (signups from social)

### Paid Ads:
- Impressions
- Clicks
- CTR (click-through rate)
- CPC (cost per click)
- Conversions (signups)
- CAC (customer acquisition cost)

### Email:
- Open rate (target: 25-35%)
- Click rate (target: 3-5%)
- Unsubscribe rate (keep <1%)

### Website:
- Traffic (Google Analytics)
- Bounce rate (keep <70%)
- Time on site (target: 2+ minutes)
- Conversions (signups)
